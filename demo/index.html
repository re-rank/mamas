RAG 챗봇 위젯 데모

아래 HTML 코드만 삽입하면 어떤 웹사이트에서도 바로 사용할 수 있는 AI 챗봇 위젯입니다. 실제로 동작하는 모습을 확인하기 위해 오른쪽 하단의 버튼을 눌러보세요.

설치 방법

번들 파일을 호스팅한 뒤 아래 스니펫만 삽입하면 됩니다. apiUrl을 여러분의 백엔드 엔드포인트로 바꿔주세요. 

<script src="/path/to/widget.js"></script>
<script>
  window.ChatWidget.init({
    apiUrl: 'https://your-backend.example.com/chat',
    title: 'AI 상담원',
    subtitle: '24시간 언제든지 물어보세요'
  });
</script>

기능 요약

  • Shadow DOM 기반으로 외부 스타일과 완벽 분리
  • HTTP API 호출 후 단건 응답 방식 RAG 연동
  • 초기 메시지, 테마 컬러 등 손쉬운 커스터마이징
